A pending/placeholder marker — an amber dot + label. Use for any value that is not yet real, so it reads as obviously unfinished.

```jsx
CAGE code <PlaceholderBadge>Pending</PlaceholderBadge>
Mailing address <PlaceholderBadge>Published at launch</PlaceholderBadge>
```

Defaults to "Pending" if no children are passed.
