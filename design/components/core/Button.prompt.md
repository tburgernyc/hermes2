The BurgerGov pill button — use for any action; `primary` for the single most important action on a surface.

```jsx
<Button variant="primary">Approve sourcing</Button>
<Button variant="secondary">View capability statement</Button>
<Button variant="ghost" size="sm">Reject</Button>
<Button variant="danger">Delete</Button>
```

Variants: `primary` (gradient navy, default), `secondary` (frosted glass), `ghost` (quiet outline), `danger` (solid red). Sizes: `sm`, `md` (default). `block` stretches to full width. All native button attrs (`type`, `disabled`, `onClick`) pass through.
