The signature frosted-glass surface. Use to group any related content.

```jsx
<Card>
  <h3>Custom software development</h3>
  <p>Full-stack systems built to spec.</p>
</Card>

<Card as="li" size="sm" interactive>…</Card>
```

`size`: `md` (default, marketing) or `sm` (denser console rows). `interactive` adds the hover lift. `as` sets the tag.
